# NoteList

Hello, this project is a basic Task List tool, enabling writing down, marking and deleting tasks. A nice tool for keeping track of your daily tasks on any device (well, except a smartwatch).

## Used technologies:

- HTML
- BEM
- CSS
- Flex
- Grid 
- JS
- npm
